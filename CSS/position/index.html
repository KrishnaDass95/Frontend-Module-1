CSS Positions

  1. Static: default position of element
  2. Relative: Position an element relative to its default position
  3. Abolute: Element is positioned relative to its parent. If parent is not present, then element is positioned relative to its main document. Element is removed from the normal document flow, no space is created for that element.
  4. Fixed: elements is positioned relative to its viewport(viewport is the document you view as you scroll). Element does not move, it stays in the same position even while scrolling. Like the nav bar in many webpages. Its fixed. Element is removed in the document flow.
  5. Sticky: same as fixed, till the element hits the top, it has normal position, after it touches the top, then its position becomes fixed.
